# DB Tuning Lab

A hands-on database tuning lab built with Laravel and MariaDB.

This repository is used to experiment with database design, indexing strategies, query optimization, and performance tuning techniques in a practical environment.

## Goals

* Learn database tuning through real experiments
* Understand how indexes affect query performance
* Analyze execution plans using EXPLAIN
* Compare different query and schema designs
* Validate concepts learned from database design books

## Topics

* Single-column indexes
* Composite indexes
* Covering indexes
* Query optimization
* EXPLAIN analysis
* ORDER BY optimization
* GROUP BY optimization
* Large dataset performance testing

## Environment

* Laravel
* MariaDB
* Docker

## Planned Experiments

* Generate datasets with 100,000+ records
* Benchmark indexed vs non-indexed queries
* Compare execution plans
* Measure query performance under different schema designs

## Purpose

This project is intended as a personal learning environment for studying database internals and performance optimization techniques.
